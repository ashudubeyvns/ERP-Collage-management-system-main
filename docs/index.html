🏫 College ERP System

A comprehensive Java Swing-based College ERP module with modern glassmorphism UI design, featuring attendance tracking, assignment management, sessional marks, faculty management, and reporting — with full dark mode support.

✨ Features

Professional-grade academic management system with modern UI

🔐

Secure Login

Blue gradient background with glassmorphism login card, password show/hide, remember me, role-based authentication (Admin, Faculty, Student).

📊

Dashboard Analytics

6 stat cards with animated charts — attendance trend line chart, department-wise pie chart, faculty workload bar chart.

🌙

Dark Mode

Full dark theme with #111827 background, #1F2937 cards, #3B82F6 primary accent. Toggle from top bar or settings.

✅

Mark Attendance

Subject/date/section selector, student list with Present/Absent/Late toggle, Mark All buttons, toast notifications.

📝

Assignments

Upload marks, manage max marks, update records with subject-wise sections and action buttons.

📋

Sessionals

Enter Sessional-1, 2, 3 marks with auto-calculated totals and Pass/Fail status.

📄

Reports & Export

Export to PDF via print, Excel CSV with BOM, and Print with formatted headers and color-coded data.

🎓

Multi-Course

BCA, MCA, B.Tech CSE/IT, B.Sc CS, M.Sc CS — 6 courses with 30+ semester-wise subjects.

🔔

Toast Notifications

Animated slide-in success/error/warning/info notifications replacing JOptionPane dialogs.

🎬

Splash & Loading

Branded splash screen with logo, version info, animated loading progress bar.

🎨

Modern UI Components

Glassmorphism panels, ripple buttons, sticky-header tables with search/sort/filter, animated charts.

🌐

REST API

Built-in HTTP server on port 8080 — health check, session info, course list, attendance records.

🚀 Getting Started

Ready to use it? Here's how

1

Download the JAR

Grab the latest release from GitHub Releases.

2

Run it

Make sure you have JDK 26+ installed, then run:

java -jar AttendanceMgt.jar
3

Or build from source

git clone https://github.com/ashudubeyvns/attendance-management-system.git
cd attendance-management-system/AttendanceMgt_1 && javac -d build/classes src/attendancemgt/*.java && java -cp build/classes attendancemgt.AttendanceMgt
4
